This site is now configured as a blog-focused Jekyll site.

Use this homepage for a short introduction, bio, or a note about what you write about.

## What to edit

- Update site-wide settings in `_config.yml`.
- Update the sidebar profile details in `_config.yml` under `author`.
- Add or edit blog posts in `_posts/`.
- Adjust the top navigation in `_data/navigation.yml`.

## Writing posts

Create files in `_posts/` using the `YYYY-MM-DD-title.md` format. Posts will automatically appear in the blog archive.

## Hosting

This repository can still be hosted with GitHub Pages in the usual way.
